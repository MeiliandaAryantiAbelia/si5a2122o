Hasil kesimpulan pertemuan 4 yang saya ambil adalah:

  1. Page/laman/halaman itu mirip dengan post, tetapi berbeda peruntukannya
  2. Page diperuntukkan untuk konten yang bersifat "tetap" atau jarang sekali di update, contohnya: visi misi, struksut organisasi, tentang perusahaan, kontak, lokasi => kesemuanya mungkin diupdate, akan tetapi sangat jarang di update
  3. Bedanya page sama pos adalah kalo page tidak ada kategorinya
  4. Cara membuat page adalah pilih laman tambah baru kemudian buat kontennya dan terbitkan
  5. Berbeda dengan pos, pos sangat sering akan berubah dan tidak menjadi viral lagi jika sudah ada pos lainnya (menimpa)
  6. Pos digunakan untuk yang sifatnya terbarukan/ dapat digantikan dengan pos baru lainnya
  7. Cara membuat pos adalah pilih pos tambah baru kemudian buat kontennya terus ke pengaturan yang pos buat kategori dan terbitkan
  8. Cara membuat custom menu:
    9. - Pilih akses Tampilan -> Menu
    - Pastikan memilih menu utama horizontal, kemudian klik Pilih
    - Jika mau tambah menu tinggal pilih saja dan klik tambahkan ke menu dan atur menu nya sesuai yang diinginkan
    Tampilan hasil custom menu

    Tampilan menu di front end

  9. Jika mau menambahkan menu berupa link tinggal pilih taut tersuai kemudian isi URL dan Teks Tautan, dan simpan Menu
  10. Cara install plugin classic widget:
    11. - Pilih akses tampilan -> Widget
    - Kemudian download classic widget.
    - Taruh file download di folder mana saja, gak perlu di extract, lokasinya diingat2.
    - Klik menu Plugin -> Tambah baru.
    - Kemudian klik Unggah Plugin, nah pilih file zip yang kita download tadi.
    - Klik Instal sekarang dan aktifkan.

  12. Untuk versi wordpress terbaru, widget nya memang ada perubahan pemakaian (menggunakan blok) sedangkan wordpress versi sebelumnya dipakai dengan drag drop
  13. Jika tema nya beda, nanti penempatan lokasi widgetnya juga akan berbeda.
  14. Caranya:
    15. - Klik menu Tampilan -> Tema, tambah baru
    - Cari tema yang keren seperti Spacious
    - Klik Instal, kemudian aktifkan
    - Kemudian klik Mulai Spacious dan cari tema yang gratisan
    - Klik Preview terlebih dahulu utk cek demo
    - Jika sudah jatuh cinta dengan tema yang dipilih, klik Import Demo, klik confirm dan tunggu sampai selesai
